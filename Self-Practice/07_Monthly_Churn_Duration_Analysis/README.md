## 📝 Challenge

```text
The Product team wants to monitor monthly user churn.

A churned user is defined as a user who has made at least one
trip historically but has no trips in the current month.

For each month, calculate the number of churned users and
segment them by churn duration:

- Inactive for 1 month
- Inactive for 2 months
- Inactive for 3 months
- Inactive for more than 3 months

Output:

month | churn_1_month | churn_2_month | churn_3_month | churn_over_3_month | total_churn
```
