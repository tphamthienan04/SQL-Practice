# SQL Problem Solving Portfolio

> A collection of SQL solutions to interview-style challenges from **DataLemur** and **LeetCode**, demonstrating practical SQL skills for Data Analyst, Business Analyst, and Analytics Engineer roles.

## 📖 About

This repository documents my SQL learning journey through real-world and interview-focused problems. The challenges cover essential SQL concepts, analytical thinking, and query optimization techniques commonly assessed during technical interviews.

## 🛠️ Skills Covered

- SQL Querying
- Data Filtering & Sorting
- Aggregate Functions
- GROUP BY & HAVING
- CASE WHEN
- INNER, LEFT & SELF JOIN
- Common Table Expressions (CTEs)
- Window Functions
- Ranking Functions
- Date & Time Functions
- String Functions
- Subqueries
- Data Cleaning & Transformation
- Analytical Problem Solving
